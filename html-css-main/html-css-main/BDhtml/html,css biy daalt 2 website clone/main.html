Mission Partnership
SpaceX will launch Haven-1 on a Falcon 9 rocket and transport the crew using its Dragon spacecraft. Haven-1 will feature Starlink - SpaceX's high-speed internet— providing Gigabit speeds and low-latency connectivity via laser terminals to the Starlink satellite network. The use of Starlink on Haven-1 will mark the first deployment of Starlink internet on a commercial space station.
